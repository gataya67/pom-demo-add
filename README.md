# Cypress POM Demo App

A small React + Cypress project demonstrating Page Object Model automation.

## Tech Stack

- React / Vite
- Cypress
- Page Object Model
- Fixtures
- API mocking with cy.intercept()
- GitHub Actions CI
- Cypress screenshots/videos as CI artifacts

## Test Coverage

- Smoke test
- Valid login
- Invalid login
- Add item to cart
- Checkout flow
- Mocked API success
- Mocked API failure

## Run App Locally

```bash
npm install
npm run dev
